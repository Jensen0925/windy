# China locations

全国省、市、区县三级行政区划搜索索引，不含乡镇和街道。

## Coverage

- 数据版本：`2025.251231.260403`
- 数据更新时间：`2026-04-03`
- 记录数：3,635（省级 34、地级 392、区县级 3,209）
- 搜索字段：中文简称、正式名称、完整路径、拼音、拼音首字母、行政区划代码
- 坐标：WGS84；原始 GCJ-02 坐标已在生成时转换，台湾缺失坐标使用旧版公开坐标补齐

## Source

行政区划、拼音和主要坐标来自
[AreaCity-JsSpider-StatsGov](https://github.com/xiangyuecn/AreaCity-JsSpider-StatsGov)，版本
`2025.251231.260403`。上游数据整合国家地名信息库、高德地图行政区域和腾讯地图行政区划。
使用和再分发时应同时遵守上游项目及其原始数据源的许可和服务条款。

此包只提交搜索所需的中心点，不包含行政区边界 polygon。
